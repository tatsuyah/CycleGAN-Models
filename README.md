## Prerequisits
 * CycleGAN (See the [README](https://github.com/junyanz/CycleGAN) for installation)

## Usage
```
mv ./model_dir/ path/to/models_dir/
cd path/to/cyclegan-root/
DATA_ROOT=/path/to/data/ name=model_dir th train.lua
```
